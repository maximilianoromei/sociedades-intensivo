  Integrar la LGS con el Código Civil y Comercial.

El curso sigue el orden lógico de la Ley General de Sociedades.

Primero:

\begin{itemize}
\tightlist
\item
  teoría general.
\end{itemize}

Luego:

\begin{itemize}
\tightlist
\item
  tipos societarios.
\end{itemize}

Finalmente:

\begin{itemize}
\tightlist
\item
  reorganización,
\item
  disolución,
\item
  liquidación.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{¿Qué estudia el Derecho
Societario?}\label{quuxe9-estudia-el-derecho-societario}

Es la rama del Derecho Privado que regula:

\begin{itemize}
\tightlist
\item
  constitución;
\item
  funcionamiento;
\item
  organización;
\item
  extinción de las sociedades.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{¿Por qué existen las
sociedades?}\label{por-quuxe9-existen-las-sociedades}

Porque una persona sola muchas veces no puede:

\begin{itemize}
\tightlist
\item
  reunir capital;
\item
  asumir riesgos;
\item
  organizar grandes emprendimientos.
\end{itemize}

La sociedad permite superar esas limitaciones.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Sociedad y actividad
económica}\label{sociedad-y-actividad-econuxf3mica}

Las sociedades son instrumentos jurídicos.

No constituyen un fin.

Constituyen un medio para desarrollar actividades económicas.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Importancia económica}\label{importancia-econuxf3mica}

Las sociedades permiten:

\begin{itemize}
\tightlist
\item
  concentración de capital;
\item
  especialización;
\item
  continuidad de la empresa;
\item
  limitación del riesgo.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Evolución histórica}\label{evoluciuxf3n-histuxf3rica}

La sociedad moderna es resultado de una evolución que comienza:

\begin{itemize}
\tightlist
\item
  Derecho romano.
\item
  Edad Media.
\item
  Revolución Industrial.
\item
  Capitalismo contemporáneo.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Derecho romano}\label{derecho-romano}

Las formas asociativas romanas carecían de autonomía semejante a la
actual.

Predominaba la relación personal entre los integrantes.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Edad Media}\label{edad-media}

Durante la Edad Media aparecen:

\begin{itemize}
\tightlist
\item
  compañías familiares;
\item
  sociedades marítimas;
\item
  commenda;
\item
  corporaciones.
\end{itemize}

Estas figuras anticipan el derecho societario moderno.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Revolución Industrial}\label{revoluciuxf3n-industrial}

La Revolución Industrial produjo:

\begin{itemize}
\tightlist
\item
  enormes inversiones;
\item
  necesidad de grandes capitales;
\item
  separación entre propiedad y administración.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Sociedad moderna}\label{sociedad-moderna}

Actualmente la sociedad constituye:

\begin{itemize}
\tightlist
\item
  una organización;
\item
  con patrimonio propio;
\item
  destinada a desarrollar actividad económica.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Derecho Comercial y
sociedades}\label{derecho-comercial-y-sociedades}

El Derecho Societario integra el Derecho Comercial.

Regula uno de los instrumentos fundamentales de la actividad empresaria.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Empresa}\label{empresa}

La empresa es:

Una organización de factores productivos destinada a producir bienes o
servicios.

No constituye una persona jurídica.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Sociedad}\label{sociedad}

La sociedad es:

Una persona jurídica creada conforme a la ley para organizar una empresa
o desarrollar una actividad económica.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Empresa y sociedad}\label{empresa-y-sociedad}

No son sinónimos.

Empresa:

fenómeno económico.

Sociedad:

instrumento jurídico.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Sociedad y empresario}\label{sociedad-y-empresario}

Es posible que exista:

\begin{itemize}
\tightlist
\item
  empresa sin sociedad;
\item
  sociedad sin empresa (supuestos excepcionales);
\item
  empresario individual;
\item
  empresario social.
\end{itemize}

No existe identidad conceptual.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Empresa y establecimiento}\label{empresa-y-establecimiento}

La empresa tampoco se identifica con el establecimiento.

Empresa:

actividad organizada.

Establecimiento:

conjunto de bienes afectados a esa actividad.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Empresa y patrimonio}\label{empresa-y-patrimonio}

El patrimonio pertenece al sujeto.

La empresa no es sujeto.

La sociedad sí.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Relación entre empresa y
sociedad}\label{relaciuxf3n-entre-empresa-y-sociedad}

La sociedad constituye el vehículo jurídico que normalmente organiza la
empresa.

La empresa constituye la actividad.

La sociedad constituye el sujeto de derecho.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Descentralización
empresaria}\label{descentralizaciuxf3n-empresaria}

El crecimiento empresario exige mecanismos de descentralización.

Entre ellos:

\begin{itemize}
\tightlist
\item
  sucursal;
\item
  agencia;
\item
  filial.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Descentralización}\label{descentralizaciuxf3n}

Permite:

\begin{itemize}
\tightlist
\item
  expansión territorial;
\item
  especialización;
\item
  disminución de costos;
\item
  organización eficiente.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Sucursal}\label{sucursal}

La sucursal:

\begin{itemize}
\tightlist
\item
  carece de personalidad jurídica propia;
\item
  integra el patrimonio de la casa matriz;
\item
  desarrolla actividades permanentes.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Características de la
sucursal}\label{caracteruxedsticas-de-la-sucursal}

Presenta:

\begin{itemize}
\tightlist
\item
  cierta autonomía funcional;
\item
  administración propia;
\item
  dependencia jurídica respecto de la sociedad principal.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Agencia}\label{agencia}

La agencia constituye un centro de negocios de menor autonomía.

Normalmente:

\begin{itemize}
\tightlist
\item
  promociona;
\item
  intermedia;
\item
  representa.
\end{itemize}

No constituye un sujeto distinto.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Diferencias entre sucursal y
agencia}\label{diferencias-entre-sucursal-y-agencia}

\subsection{Sucursal}\label{sucursal-1}

\begin{itemize}
\tightlist
\item
  actividad permanente;
\item
  mayor autonomía;
\item
  organización estable.
\end{itemize}

\subsection{Agencia}\label{agencia-1}

\begin{itemize}
\tightlist
\item
  funciones limitadas;
\item
  menor autonomía;
\item
  actuación representativa.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Filial}\label{filial}

La filial sí constituye una sociedad distinta.

Posee:

\begin{itemize}
\tightlist
\item
  personalidad propia;
\item
  patrimonio propio;
\item
  órganos propios.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Filial}\label{filial-1}

Aunque exista control económico, la filial conserva personalidad
jurídica independiente.

No existe confusión automática de patrimonios.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Grupo empresario}\label{grupo-empresario}

Un grupo empresario supone:

\begin{itemize}
\tightlist
\item
  pluralidad de sociedades;
\item
  dirección económica común;
\item
  autonomía jurídica de cada integrante.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Independencia jurídica}\label{independencia-juruxeddica}

Cada sociedad responde:

\begin{itemize}
\tightlist
\item
  por su patrimonio;
\item
  por sus obligaciones;
\item
  por sus órganos.
\end{itemize}

La existencia del grupo no elimina esa autonomía.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Introducción al régimen
legal}\label{introducciuxf3n-al-ruxe9gimen-legal}

La regulación principal surge de:

\begin{itemize}
\tightlist
\item
  Ley General de Sociedades;
\item
  Código Civil y Comercial.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Evolución legislativa}\label{evoluciuxf3n-legislativa}

Principales hitos:

\begin{itemize}
\tightlist
\item
  Código de Comercio.
\item
  Ley 19.550.
\item
  Ley 22.903.
\item
  Ley 26.994.
\item
  Reformas posteriores.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Ley 19.550}\label{ley-19.550}

La Ley General de Sociedades constituye el eje del Derecho Societario
argentino.

Su estructura permanece vigente con numerosas reformas.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Finalidad de la LGS}\label{finalidad-de-la-lgs}

La ley procura:

\begin{itemize}
\tightlist
\item
  seguridad jurídica;
\item
  organización;
\item
  protección de terceros;
\item
  funcionamiento eficiente de las sociedades.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Principios orientadores}\label{principios-orientadores}

Entre ellos:

\begin{itemize}
\tightlist
\item
  personalidad jurídica;
\item
  tipicidad;
\item
  publicidad;
\item
  registración;
\item
  especialidad.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Estructura de la ley}\label{estructura-de-la-ley}

La ley regula:

\begin{itemize}
\tightlist
\item
  teoría general;
\item
  funcionamiento;
\item
  administración;
\item
  reorganización;
\item
  disolución;
\item
  liquidación;
\item
  tipos sociales.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Reforma de la Ley 26.994}\label{reforma-de-la-ley-26.994}

La unificación del Derecho Privado produjo profundas modificaciones.

Especialmente respecto de:

\begin{itemize}
\tightlist
\item
  personalidad jurídica;
\item
  sociedades no constituidas regularmente;
\item
  unipersonalidad.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Influencia del Código Civil y
Comercial}\label{influencia-del-cuxf3digo-civil-y-comercial}

Desde 2015 la LGS debe interpretarse conjuntamente con el CCyC.

No existen compartimentos estancos.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Aplicación supletoria}\label{aplicaciuxf3n-supletoria}

Cuando la LGS no regula una cuestión determinada:

corresponde acudir al Código Civil y Comercial.

Siempre que no exista incompatibilidad.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Personas jurídicas}\label{personas-juruxeddicas}

El Código Civil y Comercial incorpora una teoría general de las personas
jurídicas.

Las sociedades integran esa categoría.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Artículo 2 LGS}\label{artuxedculo-2-lgs}

La sociedad es sujeto de derecho.

Esta disposición constituye uno de los pilares del régimen societario
argentino.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Alcance del artículo 2}\label{alcance-del-artuxedculo-2}

El reconocimiento de personalidad implica:

\begin{itemize}
\tightlist
\item
  patrimonio propio;
\item
  capacidad;
\item
  responsabilidad propia;
\item
  individualidad jurídica.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Consecuencia práctica}\label{consecuencia-pruxe1ctica}

Los bienes sociales pertenecen a la sociedad.

No pertenecen a los socios individualmente considerados.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Patrimonio separado}\label{patrimonio-separado}

Existe separación entre:

\begin{itemize}
\tightlist
\item
  patrimonio social;
\item
  patrimonio de cada socio.
\end{itemize}

Este principio resulta esencial para comprender el régimen societario.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Capacidad}\label{capacidad}

La sociedad puede:

\begin{itemize}
\tightlist
\item
  contratar;
\item
  adquirir bienes;
\item
  comparecer en juicio;
\item
  asumir obligaciones.
\end{itemize}

Siempre dentro de su objeto y del régimen legal aplicable.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Nombre}\label{nombre}

La sociedad posee un nombre propio.

Puede consistir en:

\begin{itemize}
\tightlist
\item
  razón social;
\item
  denominación social.
\end{itemize}

Según el tipo elegido.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Domicilio}\label{domicilio}

Toda sociedad posee domicilio.

El domicilio determina múltiples consecuencias jurídicas.

Entre ellas:

\begin{itemize}
\tightlist
\item
  competencia;
\item
  registración;
\item
  publicidad.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Nacionalidad}\label{nacionalidad}

La personalidad jurídica no debe confundirse con la cuestión de la
nacionalidad.

Se trata de problemas diferentes.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Personalidad jurídica}\label{personalidad-juruxeddica}

La personalidad constituye una técnica jurídica.

No describe una realidad física.

Es un reconocimiento normativo.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Función de la
personalidad}\label{funciuxf3n-de-la-personalidad}

Permite:

\begin{itemize}
\tightlist
\item
  imputar derechos;
\item
  imputar obligaciones;
\item
  organizar patrimonios;
\item
  facilitar la actividad económica.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Autonomía patrimonial}\label{autonomuxeda-patrimonial}

Sin personalidad jurídica no existiría patrimonio separado.

La autonomía patrimonial constituye una consecuencia directa del
reconocimiento legal.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Responsabilidad}\label{responsabilidad}

Como principio:

la sociedad responde con su patrimonio.

Los socios responderán únicamente en los casos previstos por la ley.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Importancia práctica}\label{importancia-pruxe1ctica}

La personalidad jurídica explica:

\begin{itemize}
\tightlist
\item
  la responsabilidad;
\item
  la representación;
\item
  la actuación procesal;
\item
  la continuidad del ente.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Doctrina}\label{doctrina}

Nissen y Vítolo coinciden en destacar que la personalidad jurídica
constituye uno de los ejes estructurales del sistema societario
argentino, aunque desarrollan distintos enfoques respecto de algunos de
sus alcances frente al abuso de la forma societaria.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Presupuestos de la personalidad
jurídica}\label{presupuestos-de-la-personalidad-juruxeddica}

La personalidad jurídica requiere:

\begin{itemize}
\tightlist
\item
  constitución conforme a la ley;
\item
  organización;
\item
  reconocimiento normativo.
\end{itemize}

No depende de la voluntad de terceros.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Nacimiento de la
personalidad}\label{nacimiento-de-la-personalidad}

Como regla general, la personalidad surge con la constitución de la
sociedad conforme al régimen legal.

La inscripción produce importantes efectos frente a terceros.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Persona jurídica y socios}\label{persona-juruxeddica-y-socios}

La sociedad no constituye una suma de socios.

Es un sujeto distinto.

Por ello:

\begin{itemize}
\tightlist
\item
  posee patrimonio propio;
\item
  posee intereses propios;
\item
  posee responsabilidad propia.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Permanencia de la
personalidad}\label{permanencia-de-la-personalidad}

El ingreso o retiro de socios no modifica la identidad de la persona
jurídica.

La sociedad continúa siendo el mismo sujeto.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Extinción de la
personalidad}\label{extinciuxf3n-de-la-personalidad}

La personalidad no desaparece inmediatamente con la disolución.

Subsiste durante la etapa de liquidación para concluir los negocios
pendientes.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Personalidad y órganos}\label{personalidad-y-uxf3rganos}

La sociedad actúa mediante órganos.

No actúa mediante representación convencional.

La actuación orgánica constituye actuación de la propia persona
jurídica.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Introducción a la
desestimación}\label{introducciuxf3n-a-la-desestimaciuxf3n}

La personalidad jurídica no constituye un principio absoluto.

Puede ser dejada de lado en supuestos excepcionales.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Prescindencia de la
personalidad}\label{prescindencia-de-la-personalidad}

También denominada:

\begin{itemize}
\tightlist
\item
  desestimación;
\item
  inoponibilidad;
\item
  levantamiento del velo societario.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Fundamento}\label{fundamento}

La personalidad jurídica no puede utilizarse:

\begin{itemize}
\tightlist
\item
  para defraudar;
\item
  para eludir la ley;
\item
  para perjudicar terceros;
\item
  para frustrar derechos.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Artículo 54 LGS}\label{artuxedculo-54-lgs}

La Ley General de Sociedades contempla expresamente la posibilidad de
declarar inoponible la personalidad jurídica cuando sea utilizada con
fines extrasocietarios o para violar la ley, el orden público o la buena
fe.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Artículo 144 CCyC}\label{artuxedculo-144-ccyc}

El Código Civil y Comercial también regula la inoponibilidad de la
personalidad jurídica.

Ambas normas deben interpretarse armónicamente.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Carácter excepcional}\label{caruxe1cter-excepcional}

La desestimación constituye un remedio excepcional.

No puede aplicarse por mera conveniencia.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Regla general}\label{regla-general}

La regla continúa siendo:

respeto de la personalidad jurídica.

La excepción exige circunstancias calificadas.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Abuso de la personalidad}\label{abuso-de-la-personalidad}

Existe abuso cuando la sociedad deja de ser utilizada para los fines
propios del instituto y se transforma en un instrumento para alcanzar
resultados ilícitos o abusivos.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Finalidad del instituto}\label{finalidad-del-instituto}

No procura eliminar la personalidad jurídica.

Procura impedir su utilización abusiva.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Consecuencias}\label{consecuencias}

Declarada la inoponibilidad:

los efectos de determinados actos pueden extenderse a quienes utilizaron
abusivamente la sociedad.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Alcance}\label{alcance}

La desestimación:

\begin{itemize}
\tightlist
\item
  no disuelve la sociedad;
\item
  no elimina su personalidad;
\item
  no modifica su existencia.
\end{itemize}

Simplemente impide invocarla en el caso concreto.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Interpretación
restrictiva}\label{interpretaciuxf3n-restrictiva}

La doctrina sostiene que debe aplicarse con extrema prudencia.

La personalidad jurídica constituye uno de los pilares del sistema
societario.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Tipicidad}\label{tipicidad}

Otro principio estructural de la LGS es la tipicidad.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Concepto}\label{concepto}

Tipicidad significa que la sociedad debe adoptar uno de los tipos
previstos por el legislador.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Finalidad de la tipicidad}\label{finalidad-de-la-tipicidad}

La tipicidad procura:

\begin{itemize}
\tightlist
\item
  seguridad jurídica;
\item
  previsibilidad;
\item
  protección de terceros.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Tipo y contrato}\label{tipo-y-contrato}

No deben confundirse.

El contrato social expresa la voluntad de los socios.

El tipo constituye el modelo legal elegido.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Tipos previstos}\label{tipos-previstos}

La LGS regula diversos tipos societarios.

Cada uno posee:

\begin{itemize}
\tightlist
\item
  estructura;
\item
  responsabilidad;
\item
  órganos;
\item
  funcionamiento propios.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Rasgos tipificantes}\label{rasgos-tipificantes}

Cada tipo posee elementos esenciales cuya ausencia puede impedir su
configuración.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Función práctica}\label{funciuxf3n-pruxe1ctica}

La tipicidad permite conocer anticipadamente:

\begin{itemize}
\tightlist
\item
  régimen de responsabilidad;
\item
  forma de administración;
\item
  régimen de gobierno;
\item
  fiscalización.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Reforma de la Ley 26.994}\label{reforma-de-la-ley-26.994-1}

La reforma flexibilizó parcialmente el principio de tipicidad.

No obstante, éste continúa siendo un eje del sistema. La reforma no
eliminó el principio de tipicidad.

Los tipos sociales previstos por la LGS continúan siendo de número
cerrado.

Sin embargo,

la Ley 26.994 modificó profundamente las consecuencias derivadas de la
falta de adecuación a un tipo legal.

La creación del régimen de la Sección IV sustituyó el antiguo sistema
sancionatorio por un régimen más flexible para las sociedades no
constituidas conforme a los tipos previstos por la ley.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Clasificación de las
sociedades}\label{clasificaciuxf3n-de-las-sociedades}

Las sociedades pueden clasificarse conforme a diversos criterios.

No existe una única clasificación.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Según la responsabilidad}\label{seguxfan-la-responsabilidad}

Pueden distinguirse:

\begin{itemize}
\tightlist
\item
  responsabilidad limitada;
\item
  responsabilidad ilimitada;
\item
  responsabilidad mixta.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Según la participación del
capital}\label{seguxfan-la-participaciuxf3n-del-capital}

Se distinguen:

\begin{itemize}
\tightlist
\item
  sociedades de personas;
\item
  sociedades de capital;
\item
  sociedades mixtas.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Según la organización}\label{seguxfan-la-organizaciuxf3n}

Algunos tipos presentan una organización predominantemente personal.

Otros privilegian la organización del capital.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Importancia de las
clasificaciones}\label{importancia-de-las-clasificaciones}

Las clasificaciones permiten comprender:

\begin{itemize}
\tightlist
\item
  la lógica del sistema;
\item
  la función económica de cada tipo;
\item
  las diferencias de régimen.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Nacionalidad de las
sociedades}\label{nacionalidad-de-las-sociedades}

La cuestión de la nacionalidad ha dado lugar a importantes debates
doctrinarios.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Problema jurídico}\label{problema-juruxeddico}

La pregunta consiste en determinar:

¿de qué Estado es una sociedad?

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Principales doctrinas}\label{principales-doctrinas}

Se desarrollaron, entre otras:

\begin{itemize}
\tightlist
\item
  teoría de la constitución;
\item
  teoría del domicilio;
\item
  teoría del control.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Teoría de la
constitución}\label{teoruxeda-de-la-constituciuxf3n}

La nacionalidad depende del Estado conforme a cuya legislación fue
constituida la sociedad.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Teoría del domicilio}\label{teoruxeda-del-domicilio}

La nacionalidad deriva del domicilio social efectivo.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Teoría del control}\label{teoruxeda-del-control}

La nacionalidad depende de quién ejerce el control económico de la
sociedad.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Derecho argentino}\label{derecho-argentino}

El régimen argentino adopta soluciones propias respecto de las
sociedades constituidas en el extranjero y de la ley aplicable.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Artículo 1 LGS}\label{artuxedculo-1-lgs}

La Ley General de Sociedades comienza definiendo el contrato de
sociedad.

Esta disposición constituye el punto de partida de toda la materia.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Elementos del artículo 1}\label{elementos-del-artuxedculo-1}

De su lectura surgen:

\begin{itemize}
\tightlist
\item
  pluralidad (con las excepciones legales);
\item
  aportes;
\item
  organización;
\item
  participación en resultados;
\item
  actividad económica.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Naturaleza jurídica}\label{naturaleza-juruxeddica}

La doctrina ha debatido extensamente la naturaleza del acto
constitutivo.

Las posiciones principales lo consideran:

\begin{itemize}
\tightlist
\item
  contrato;
\item
  contrato plurilateral de organización;
\item
  acto complejo de organización.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Naturaleza del acto
constitutivo}\label{naturaleza-del-acto-constitutivo}

La doctrina ha intentado explicar jurídicamente el nacimiento de la
sociedad.

Las principales posiciones son:

\begin{itemize}
\tightlist
\item
  teoría contractual;
\item
  teoría del acto colectivo;
\item
  teoría del acto complejo;
\item
  teoría institucional.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Teoría contractual}\label{teoruxeda-contractual}

Es la concepción clásica.

Considera que la sociedad nace de un contrato celebrado entre dos o más
personas.

Su fundamento radica en la existencia del consentimiento de los socios.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Fundamentos}\label{fundamentos}

Según esta teoría:

\begin{itemize}
\tightlist
\item
  existe acuerdo de voluntades;
\item
  cada socio asume obligaciones;
\item
  el contrato genera derechos recíprocos;
\item
  la sociedad nace por el consentimiento.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Críticas}\label{cruxedticas}

La teoría contractual explica adecuadamente el nacimiento de las
sociedades primitivas.

Sin embargo,

presenta dificultades para justificar:

\begin{itemize}
\tightlist
\item
  la continuidad de la sociedad pese al cambio de socios;
\item
  la existencia de órganos sociales;
\item
  la subordinación de la voluntad individual al interés social.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Teoría del acto colectivo}\label{teoruxeda-del-acto-colectivo}

Esta teoría sostiene que la sociedad surge de una declaración colectiva
de voluntad.

No existen intereses contrapuestos entre los socios.

Todos persiguen un mismo fin común.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Características}\label{caracteruxedsticas}

A diferencia del contrato tradicional,

las voluntades no se enfrentan.

Se unen para alcanzar un objetivo común:

el desarrollo de la empresa social.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Críticas}\label{cruxedticas-1}

Aunque supera algunos inconvenientes de la teoría contractual,

continúa explicando solamente el momento constitutivo.

No alcanza para justificar plenamente el funcionamiento posterior de la
sociedad.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Teoría del acto complejo}\label{teoruxeda-del-acto-complejo}

Sostiene que el nacimiento de la sociedad resulta de una pluralidad de
actos jurídicos coordinados.

No basta únicamente el consentimiento de los socios.

También intervienen:

\begin{itemize}
\tightlist
\item
  la inscripción;
\item
  el control estatal;
\item
  otros actos jurídicos necesarios para la constitución.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Características}\label{caracteruxedsticas-1}

La sociedad nace como consecuencia de un procedimiento complejo.

Cada acto integra una secuencia jurídica destinada a producir un único
resultado:

el nacimiento de la persona jurídica.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Teoría institucional}\label{teoruxeda-institucional}

Representa una evolución respecto de las teorías anteriores.

La sociedad deja de ser vista únicamente como un contrato.

Pasa a concebirse como una institución organizada para la consecución de
un fin permanente.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Características}\label{caracteruxedsticas-2}

En la sociedad existen intereses que trascienden a los socios
individuales.

La organización adquiere autonomía respecto de quienes la integran.

Cobran especial importancia:

\begin{itemize}
\tightlist
\item
  el interés social;
\item
  los órganos;
\item
  la continuidad de la empresa.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Ventajas}\label{ventajas}

La teoría institucional explica adecuadamente:

\begin{itemize}
\tightlist
\item
  la permanencia de la sociedad;
\item
  la sustitución de socios;
\item
  el funcionamiento de los órganos;
\item
  la existencia de un interés propio de la persona jurídica.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Situación actual}\label{situaciuxf3n-actual}

Ninguna teoría explica por sí sola todos los aspectos del fenómeno
societario.

La doctrina contemporánea reconoce que la sociedad presenta elementos:

\begin{itemize}
\tightlist
\item
  contractuales;
\item
  organizativos;
\item
  institucionales.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Derecho argentino}\label{derecho-argentino-1}

El artículo 1 de la Ley General de Sociedades mantiene una base
contractual.

Sin embargo,

la regulación de la organización social, de los órganos y de la
personalidad jurídica incorpora claros elementos institucionales.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Conclusión}\label{conclusiuxf3n}

La evolución doctrinaria muestra el siguiente recorrido:

\includegraphics[width=9.17in,height=0.73in]{clase01_files/figure-latex/mermaid-figure-1.png}

Cada teoría fue ampliando la explicación del fenómeno societario, desde

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Evolución doctrinaria}\label{evoluciuxf3n-doctrinaria}

Originalmente predominó la concepción contractual.

Posteriormente la doctrina advirtió que dicha explicación resultaba
insuficiente para comprender el funcionamiento permanente de la
sociedad.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Contratos de cambio}\label{contratos-de-cambio}

En los contratos de cambio:

\begin{itemize}
\tightlist
\item
  las prestaciones se enfrentan;
\item
  los intereses son contrapuestos;
\item
  cada parte procura satisfacer un interés individual.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Contratos de organización}\label{contratos-de-organizaciuxf3n}

En los contratos de organización:

\begin{itemize}
\tightlist
\item
  las prestaciones convergen;
\item
  existe finalidad común;
\item
  la organización subsiste luego de celebrado el contrato.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Importancia de la
distinción}\label{importancia-de-la-distinciuxf3n}

La sociedad no se agota con el consentimiento inicial.

Después del nacimiento:

\begin{itemize}
\tightlist
\item
  continúa funcionando;
\item
  produce decisiones;
\item
  incorpora socios;
\item
  modifica su organización.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Organización permanente}\label{organizaciuxf3n-permanente}

La organización constituye uno de los elementos característicos de la
sociedad moderna.

La SAU presenta un régimen más riguroso que la SA tradicional.

Ello procura proteger:

\begin{itemize}
\tightlist
\item
  acreedores;
\item
  tráfico jurídico;
\item
  seguridad patrimonial.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Fundamento}\label{fundamento-1}

La existencia de un único socio incrementa el riesgo de confusión
patrimonial.

Por ello el legislador establece mayores controles.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Capital}\label{capital}

La existencia de un solo socio no elimina la necesidad de un capital
social suficiente.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Patrimonio}\label{patrimonio-1}

Debe mantenerse estrictamente separado:

\begin{itemize}
\tightlist
\item
  patrimonio social;
\item
  patrimonio del socio único.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Administración}\label{administraciuxf3n-1}

La administración continúa siendo un órgano de la sociedad.

No actúa como mandatario del socio único.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Sociedad y condominio}\label{sociedad-y-condominio}

No deben confundirse.

En el condominio existe cotitularidad sobre bienes.

En la sociedad existe una persona jurídica distinta.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Sociedad y comunidad
hereditaria}\label{sociedad-y-comunidad-hereditaria}

La comunidad hereditaria tampoco constituye una sociedad.

Carece de finalidad organizativa propia del contrato social.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Sociedad y asociación}\label{sociedad-y-asociaciuxf3n-1}

La asociación persigue un fin ideal.

La sociedad procura desarrollar actividad económica organizada.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Sociedad y fundación}\label{sociedad-y-fundaciuxf3n-1}

La fundación nace de un patrimonio afectado a un fin determinado.

No posee socios.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Sociedad y fideicomiso}\label{sociedad-y-fideicomiso}

El fideicomiso constituye un patrimonio de afectación.

No genera una persona jurídica independiente.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Sociedad y contratos
asociativos}\label{sociedad-y-contratos-asociativos}

Los contratos asociativos regulados por el Código Civil y Comercial:

\begin{itemize}
\tightlist
\item
  no generan personalidad jurídica;
\item
  mantienen autonomía patrimonial de cada participante.
\end{itemize}

\section{Sociedad y empresario}\label{sociedad-y-empresario-1}

El empresario puede actuar:

\begin{itemize}
\tightlist
\item
  individualmente;
\item
  mediante una sociedad.
\end{itemize}

La empresa no exige necesariamente una sociedad.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Sociedad como técnica
jurídica}\label{sociedad-como-tuxe9cnica-juruxeddica}

La sociedad constituye una construcción jurídica destinada a facilitar
la organización económica.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Función económica}\label{funciuxf3n-econuxf3mica-1}

Permite:

\begin{itemize}
\tightlist
\item
  reunir capital;
\item
  distribuir riesgos;
\item
  profesionalizar la gestión.
\end{itemize}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\section{Función jurídica}\label{funciuxf3n-juruxeddica-1}

Permite:

\begin{itemize}
\tightlist
\item
  imputar derechos;
\item
  imputar obligaciones;
\item
  actuar en el tráfico jurídico.
\end{itemize}
